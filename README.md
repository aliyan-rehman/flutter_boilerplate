# Flutter Boilerplate

A complete Flutter starter project with:
- Firebase Auth (Email + Google)
- Firestore CRUD
- SQLite CRUD
- Provider State Management
- Custom Reusable Widgets

---

## Setup
1. Clone the repo
2. Run `flutter pub get`
3. Add your own `firebase_options.dart` using FlutterFire CLI
4. Add `google-services.json` in `android/app/`
5. Run `flutter run`

---

This boilerplate helps developers **start Flutter projects faster** with commonly used architecture already set up.
